Fade In

on-screen on-screen_fade--in

Fade Out

on-screen on-screen_fade--out

Mask In

Requires a wrapper element

on-screen on-screen_mask--in__DIRECTION

Mask Out

Requires a wrapper element

on-screen on-screen_mask--out__DIRECTION

Mask Color

Requires a wrapper element. Can be used with mask in or out. Use hex code.

on-screen on-screen_mask--in__DIRECTION on-screen_mask--color__HEX

Move In

Default is 25%.

on-screen on-screen_move--in__DIRECTION

Move Out

Default is 25%.

on-screen on-screen_move--out__DIRECTION

Move Amount

Use in combination with move in or out to specify how much the element moves. Must be a percentage. Can be more than 100%.

on-screen on-screen_move--in__DIRECTION on-screen_move--PERCENTAGE

Percent of Element

Specify the minimum percent of the element visible on screen. Can be more than 100%. Use in combination with any animation(s).

on-screen on-screen_percent--PERCENT

Transition Duration

Specify in milliseconds. The default is 500. Use in combination with any animation(s).

on-screen EFFECT on-screen_duration--DURATION

Transition Delay

Specify in milliseconds. The default is 500. Use in combination with any animation(s).

on-screen EFFECT on-screen_delay--DELAY

Don't Unanimate

By default, when the animation is complete the element will not unanimate once it's off screen. If you would like it to unanimate add the class "unanimate"

on-screen EFFECT unanimate

Combine Effects

Mix and match classes to create custom effects.
