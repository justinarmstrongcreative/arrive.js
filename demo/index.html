arrive.js

A JavaScript plugin to detect and animate elements once they arrive on screen.

Fade In

arrive arrive-fade_in

Fade Out

arrive arrive-fade_out

Mask In

Requires a wrapper element. Can mask up, right down or left.

arrive arrive-mask_in--DIRECTION

Mask Out

Requires a wrapper element. Can mask up, right down or left.

arrive arrive-mask_out--DIRECTION

Mask Color

Requires a wrapper element. Must be used in combination with mask in or out. Accepts hex code.

arrive arrive-mask_in--DIRECTION arrive-mask_color--HEX

Move In

Default is 25%. Can move up, right down or left.

arrive arrive-move_in--DIRECTION

Move Out

Default is 25%. Can move up, right down or left.

arrive arrive-move_out--DIRECTION

Move Amount

Use in combination with move in or out to specify how much the element moves. Must be a percentage. Can be more than 100%.

arrive arrive-move_in--DIRECTION arrive-move_PERCENTAGE

Percent of Element

The minimum percent of the element visible on screen. Use in combination with any animation(s). Can be more than 100%.

arrive arrive-percent_PERCENT EFFECT

Transition Delay

Specify in milliseconds. Default is 500. Use in combination with any animation(s).

arrive EFFECT arrive-delay_DELAY

Transition Duration

Specify in milliseconds. Default is 500. Use in combination with any animation(s).

arrive EFFECT arrive-duration_DURATION

Transition Timing

Default is ease. If using steps or cubic-bezier, no spaces. Use in combination with any animation(s).

arrive EFFECT arrive-timing_FUNCTION

Unanimate

Will unanimate the element once it is off screen. Allows animation to run again.

arrive arrive-unanimate EFFECT

Combine Effects

Mix and match classes to create custom effects.
