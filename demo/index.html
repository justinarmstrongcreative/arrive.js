Move Amount

Must be a percentage. Can be more than 100%.









Percent of element

Specify the minimum percent of the element visible on screen. Can be more than 100%. Use in combination with any animation(s).









Transition Duration

Specify in milliseconds. The default is 500. Use in combination with any animation(s).









Transition Delay

Specify in milliseconds. The default is 500. Use in combination with any animation(s).









Don't Unanimate

By default, when the animation is complete the element will not unanimate once it's off screen. If you would like it to unanimate add the class "unanimate"









Combine effects

All animations must either transition in or out. Fade, mask, move, up, right, bottom and left can be mixed.
